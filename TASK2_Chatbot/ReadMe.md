🛒 E-Commerce FAQ Chatbot

Description

This is a simple FAQ Chatbot for an e-commerce system. It helps users get instant answers to common shopping-related questions using NLP techniques like TF-IDF Vectorization and Cosine Similarity.

Features

* Answers e-commerce related FAQs
* NLP-based question matching
* Streamlit chat interface
* Fast and accurate responses

Technologies Used

* Python
* Streamlit
* Scikit-learn

 How to Run

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
